
# AI-Driven Fault Detection & Photovoltaic Power Prediction

## Overview

This project focuses on developing a data-driven system for **photovoltaic (PV) power forecasting** and **fault detection**.
The goal is to improve energy performance monitoring by predicting power output and identifying anomalies in real time.

The system combines machine learning models with a web-based dashboard to provide actionable insights for monitoring and decision-making.

---

## Key Features

* Power output prediction using machine learning models
* Anomaly detection for early fault identification
* Data preprocessing and time-series analysis
* Interactive dashboard for visualising predictions and system performance
* Backend API for model integration and data handling

---

## Tech Stack

* **Python** (Pandas, NumPy)
* **Machine Learning:** Scikit-learn, TensorFlow
* **Backend:** FastAPI
* **Frontend:** Next.js
* **Visualisation:** Matplotlib

---

## Methodology

### 1. Data Processing

* Cleaning and preprocessing of NASA hystorical data
* Handling missing values and feature engineering
* Time-series structuring for model training

### 2. Power Prediction

* Training time series models to forecast PV power output
* Model evaluation using standard performance metrics

### 3. Fault Detection

* Detecting anomalies based on deviations between predicted and actual values
* Identifying abnormal patterns indicating potential system faults

---

## Results

* Accurate short-term power predictions
* Ability to detect anomalies and potential faults early
* Improved visibility of system performance through dashboard visualisation
